Subnet Calculator

A Python program to calculate subnet values. It calculates total number of hosts that can be present in the given subnet, its Network Address, Broadcast Address and First & Last Valid Host IP Addresses. Additionally it displays appropriate messages for a /31 and discontinuous masks.

You can view the code on GitHub and can leave a ⭐ if you like it. 🙂

LAN Scanner

A Python program that scans a local area network and displays a list of IP and MAC Addresses of alive hosts. While the scanning is in progress, it displays a status bar to show the progress in real time.

You can view the code on GitHub and can leave a ⭐ if you like it. 🙂
